## 注册信息怎么填——别乱填，有几个地方很重要

搬瓦工的注册表单本身不复杂，但有几个地方填错了真的会有麻烦：

**国家必须真实。** 这是最重要的一条。国家选错被系统检测到，订单直接判定欺诈，轻则要求补材料，重则账户被暂停。如果你在国内，就选 China。

**邮箱要能收到邮件。** 国内常见邮箱（163、QQ、Outlook）都可以，但 Gmail 在国内收件也有时不稳定，尽量用 163 或 QQ 邮箱。邮箱关系到后续的账单通知和账户验证，不能瞎填。

**名字可以用拼音。** 搬瓦工不接受中文字符，全名填拼音就行，格式大致对就没问题，不用完全真实，但不要填一串乱码，客服会人工审核订单。

**密码建议设置得复杂一点。** 搬瓦工近两年新增了 New device authentication（新设备登录验证），每次从新设备登录会发验证码到你注册的邮箱，账户安全意识要跟上。

## 搬瓦工登录时又收不到验证码怎么办

这是另一个高频问题，和注册时的问题不一样——这次的验证码是邮件发送的，不是 Google reCAPTCHA。

搬瓦工的新设备认证机制是：用新设备登录时，系统会往你注册的邮箱发一条有效期 1 小时的验证码。国内收件有时候会有延迟，先**看一眼垃圾邮件箱**，大概有一半的概率在那里。

如果垃圾箱也没有，有三个备选方案：

- **切换到手机号验证**：如果你的账户绑定了手机号，可以选择短信验证
- **使用备用验证码（Backup Codes）**：这是需要提前在 KiwiVM 后台 Account Security 页面生成的，建议登录后就设置好
- **临时跳过验证**：搬瓦工曾短暂提供过这个选项，但不保证长期可用

说实话，最省心的办法是提前把备用验证码存好。登录账户后，进 Account Security，生成一组备用码，截图或者记在密码管理器里，以后遇到收不到邮件的情况直接用，不至于被锁在门外。
